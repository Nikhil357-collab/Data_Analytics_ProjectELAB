# Data_Analytics_ProjectELAB
# 🔍 LinkedIn Job Trend Analysis (Web Scraping Project)

## 📌 Overview
This project analyzes job trends by scraping job postings and extracting insights about:
- In-demand skills
- Job distribution across cities
- Skill vs role relationship
- Salary insights (if available)

---

## 🚀 Tech Stack
- Python
- BeautifulSoup (Web Scraping)
- Pandas (Data Analysis)
- Matplotlib / Seaborn (Visualization)

---

## 📊 Features
✔ Job data scraping  
✔ Skill extraction using NLP (keyword-based)  
✔ Heatmap visualization (skills vs city)  
✔ Skill vs Role matrix  
✔ Salary trend analysis  

---

## 📁 Dataset
Data collected using web scraping from LinkedIn public job listings.
